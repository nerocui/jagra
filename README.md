## Meteor ##
[Official Site](https://www.meteor.com/)

 - Follow the installatiion steps.
 - Pull the code base
 - Meteor has npm built in, so run
 ```
 meteor npm install
 ```
 - Then run
 ```
 meteor
 ```
 to build and delpoy to local
 - To run tests
 ```
 meteor npm run test
 ```
